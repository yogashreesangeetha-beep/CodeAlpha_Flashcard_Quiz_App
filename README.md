# Flashcard Quiz App

A simple Flashcard Quiz Application built using Python and Tkinter.

## Features
- Displays questions and answers
- Show Answer button
- Next and Previous navigation
- Add, Edit, and Delete flashcards
- Simple and clean user interface

## Technologies Used
- Python
- Tkinter

## How to Run
1. Install Python on your computer
2. Save `flashcard_app.py` in a folder
3. Open terminal or IDLE
4. Run the program using:flashcard_app.py
